# 资料分析与理解

## 目标
把用户提供的多份文件、链接、参考视频和品牌素材，先整理成一份可执行的创作理解，再进入分镜与提示词生成。

## 处理顺序
1. 先登记所有输入资源。
2. 按用户明确要求分配优先级。
3. 提取可视觉化信息、可口播信息、品牌约束、风险限制。
4. 保留冲突，不要擅自抹平。
5. 输出统一的分析摘要。

## 资源登记表
| 字段 | 说明 |
|---|---|
| source_name | 文件名或链接名 |
| source_type | 文档 / 视频 / 图片 / 网页 / 其他 |
| role | 背景 / 参考 / 品牌 / 文案 / 约束 |
| key_facts | 事实性信息 |
| visual_cues | 可视化线索 |
| confidence | 高 / 中 / 低 |

## 优先级规则
1. 用户明确指令
2. 当前项目目标
3. 品牌和业务材料
4. 参考视频结构
5. 通用宣传视频惯例

## 输出摘要应包含
- 项目目标
- 目标受众
- 核心信息点
- 视觉风格约束
- 时间长度或交付格式要求
- 明确禁区
- 待确认问题
- 需要人工推断的内容

## 关键原则
- 资料多于一个时，先做交叉验证，再汇总。
- 参考视频只用来提炼结构和节奏，不直接复制文案。
- 如果信息不足，明确标注推断项，不要伪装成事实。
- 任何与用户明确要求冲突的材料都降级处理。
