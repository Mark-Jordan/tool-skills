# 提示词示例

## 抽象到具象
| 抽象表达 | 视觉化表达 |
|---|---|
| 高效 | 多条业务流同步进入统一看板 |
| 可信 | 稳定光线、整洁空间、规范操作 |
| 智能 | 数据节点自动汇聚并形成清晰状态图 |
| 协同 | 多角色围绕同一界面协作 |
| 价值提升 | 结果面板上的关键指标明显改善 |

## 图片提示词示例
**好**
```text
现代企业运营中心，中央大屏显示多条业务流程状态，三名工作人员在屏前查看数据，中远景，35mm 镜头，略高机位，冷白屏幕光与柔和环境光，真实商业纪录片质感，底部留白，避免画面文字、随机 logo、赛博朋克特效。
```

**坏**
```text
展现高效、智能、赋能、未来感、科技感的企业宣传画面。
```

## 视频提示词示例
**好**
```text
以当前关键帧为起点，屏幕上的数据节点缓慢汇入中央状态图，工作人员轻微查看屏幕，镜头稳定缓慢推进，保持原有构图和办公空间质感，8 秒，结尾停在中央大屏与人物同框的稳定画面，不要新增场景、不要文字、不要夸张特效。
```

**坏**
```text
让画面更有科技感，表现企业未来发展和强大实力。
```

## 常见失效模式
- 词多但没有主体
- 同时写多个镜头
- 把字幕文案塞进视频提示词
- 要求模型生成清晰中文大字
- 画面风格和运镜相互冲突
